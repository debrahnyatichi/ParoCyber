# ParoCyber Assignments

This repository contains structured coursework and practical assignments completed during my Ethical Hacking & Cybersecurity Training at **ParoCyber**.

Each assignment reflects both **technical competency** and **professional, ethical, and legal standards** required in real-world cybersecurity practice.

---

## Assignments

### Assignment 1: Pentesting Agreement

This assignment focused on drafting a **real-world Penetration Testing Agreement** that defines the legal, ethical, and operational framework required before conducting any authorized security testing.

Key areas covered include:
- Scope definition and authorization
- Rules of engagement
- Testing methodology and standards
- Confidentiality and data protection
- Reporting and disclosure
- Liability and compliance considerations

### Assignment 2: Nmap & Scapy Lab

This assignment focused on hands-on reconnaissance, scanning, and packet-level analysis using **Nmap** and **Scapy**, two foundational tools in offensive and defensive cybersecurity operations.

The lab emphasized understanding how network visibility, service discovery, and packet manipulation are performed in real-world security assessments.
